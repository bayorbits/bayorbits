# Licensing

Bay Orbits is offered under a **two-track licensing model** designed for clarity:

1. **Public Track (default): GNU GPLv3**
2. **Commercial Track (paid): proprietary commercial license**

This repository uses GPLv3 as the default because Bay Orbits is a static, client-side project with no backend/SaaS service layer. GPLv3 provides clear copyleft obligations for distributed derivatives without adding additional network-use clauses beyond GPLv3 distribution requirements.

## Two-track model

### 1) Public Track — GPLv3 (default)
If you use, modify, or redistribute this project under GPLv3, you must follow GPLv3 terms.

In practice, for distributed derivative versions, this includes providing corresponding source code under GPLv3.

See full legal text in [`LICENSE`](./LICENSE).

### 2) Commercial Track — Proprietary License (paid)
If you need to use Bay Orbits in a way that does not fit GPLv3 obligations (for example, closed-source redistribution or integration into proprietary offerings), you can obtain a separate commercial license.

See summary in [`COMMERCIAL_LICENSE.md`](./COMMERCIAL_LICENSE.md).

## Which license applies to you?

- If you have **not** executed a separate commercial agreement with Bay Orbits, you are using this project under **GPLv3**.
- If you have executed a commercial agreement, your use is governed by that agreement for the scope it covers.

This is not "non-commercial vs commercial" licensing. Commercial use is allowed under GPLv3 as long as GPLv3 obligations are met.
